The problem is to place n players onto k teams. Each of the i players has a ranking r_i, the goal is to partition the players so that the teams are even. I.e. minimize the maximum mean player rank difference, pairwise, over all teams. Predefined captains and a specialized goalies adds some additional permutational complexity to the problem. This problem is known to be NP-Hard and thus heuristic methods are appropriate.

Several leagues generatable by varying the seed.
